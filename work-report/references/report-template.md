# 작업 리포트 출력 템플릿

아래 형식에 맞춰 작업 리포트를 생성한다.

---

## 작업 리포트

> 기간: {시작일} ~ {종료일}
> 작성자: {git author}

---

### 요약

- 총 커밋: N건
- 프로젝트: N개
- 주요 작업: 신규 기능 N건, 버그 수정 N건, 리팩토링 N건

---

### 이번 주 작업 내역

#### {프로젝트명 1}

**신규 기능**
- 작업 요약 (같은 주제 커밋은 하나로 통합)

**버그 수정**
- 수정 요약

**리팩토링**
- 리팩토링 요약

**기타**
- 작업 요약

#### {프로젝트명 2}

...

---

### 전주 대비 비교

| 항목 | 전주 ({전주 시작} ~ {전주 끝}) | 이번 주 ({이번주 시작} ~ {이번주 끝}) | 변화 |
|------|------|---------|------|
| 총 커밋 | N건 | N건 | +N / -N |
| 신규 기능 | N건 | N건 | +N / -N |
| 버그 수정 | N건 | N건 | +N / -N |
| 리팩토링 | N건 | N건 | +N / -N |
| 기타 | N건 | N건 | +N / -N |

**프로젝트 분포 변화**
- 전주: {프로젝트A} 위주 작업
- 이번 주: {프로젝트B} 위주 작업
- 새로 작업한 프로젝트: {프로젝트C}

**특이사항**
- 전주 대비 변화가 큰 부분 요약 (예: "버그 수정 비율 증가", "새 프로젝트 착수" 등)
